Project Components

On the main page of the Tizen native Project Wizard, you can set basic properties of a project, such as the project name, package name, location, sample, and template.

Figure: Project components

The options you can set are listed in the following table.

Table: Project component options
Option Description
Tab You can create an application based on a template or sample application.
Category tab You can create an application based on a template or sample application. If the selected category has samples or templates, they are displayed in the list on the left.
Application type Select the application type. You can preview the application in the description view on the right.
Project name Enter the name for the project to be created. The project names have the following restrictions:
  • Maximum length of 52 characters
  • Begins with a letter ([A-Za-z]) and can be followed by any number of letters, digits ([0-9]), hyphens ("-"), underscores ("_"), and periods (".").
Package name Enter the package name for the project to be created.
Default location If you select this option, the project is created in the $<workspace_location>/$<project_name> directory.
Location Manually select the location of the project. If you select the Use default location option, this option is disabled.
