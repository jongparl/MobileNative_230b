<author> Element

The <author> element represents the creator of the Tizen package. This element has no child elements.

<author> element

Represents the creator of the Tizen package.

For more information on the relationship between the elements, see the element hierarchy.

Occurrences:

  • 1 or more (optional)

Attributes:

  • author

  • email

  • website
author attribute

Name of the package creator.

Expected value:

  • Any string value
email attribute

Email of the package creator.

Expected value:

  • Any valid email ID string value
website attribute

Web site of the package creator.

Expected value:

  • Any valid Web site string value

For example:

<author author="testauthor" email="testauthor@testemail.com" website="www.testwebsite.com"/>
