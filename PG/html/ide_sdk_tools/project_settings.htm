Project Settings

In the project settings of the Tizen native Project Wizard, you can set compilation options. Currently, only the precompiled header option is provided.

Figure: Project Settings

The options you can set are listed in the following table.

Table: Project component options
Option Description
Include a precompiled header A precompiled header is a header file that is compiled into an intermediate form. Compiling can be made faster by including this file. If you select this option, the precompiled header file tizenx.h is included in your sample or template.
