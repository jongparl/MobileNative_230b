Snippets

Snippets allow you to create blocks of code that can be entered into your document by double-clicking the snippet from the Snippet Tree view. A text trigger can also be assigned to a snippet, allowing you to type the text trigger followed by a keyboard shortcut to add the snippet to your document.

To launch the Snippet Tree view, select Window > Show View > Snippet Tree View.

Creating a Snippet

Click the Create a new snippet button in the Snippet Tree view. If you want to create a package before creating a new snippet, click the Create a new snippet package button first. In the New Snippet window, fill in the following details:

  • Snippet name
  • Trigger text

    Define the text to use as an accelerator key

  • Snippet description
  • Snippet starting and closing block

    Snippet implementation, which is inserted at the current cursor position in the editor

When defining snippets, you can use dynamic snippet variables of the default form. You can define snippets as shown below:

  • When you insert the following snippet, you are prompted for the values of 'variable name' and 'default'.

    <"$${variable name}" default="$${default}"/>

  • When you insert the following snippet, the input box for the 'default' value is pre-filled with "string".

    <"$${variable name}" default="$${default:string}"/>

    Figure: Snippet with variable whose default value is shown in the input box

  • When you insert the following snippet, the input box allows you to select from the pre-suggested enumerated values.

    <"$${variable name}" default="$${default:string|boolean|date}"/>

    Figure: Snippet with variable whose pre-suggested values are available in the input box

Inserting a Snippet

To insert a snippet, you can use the Snippet Tree view or the trigger text:

  • In the Snippet Tree view, select the snippet and click the Insert the selected snippet into the document button.
  • In your code, type the trigger text and press the accelerator key on the keyboard (Ctrl + Shift + I).

If you are inserting a snippet that uses variables, you are prompted for the needed values.
