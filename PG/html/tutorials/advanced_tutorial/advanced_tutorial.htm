Advanced Tutorials

Tizen native advanced tutorials teach you how to incorporate more complex Tizen native features into your application.

You can learn much more about Tizen features and functionalities by studying the following tutorials:

  • Account Manager Features Tutorial

    This tutorial shows how you can manage user accounts and account types.

  • Application Manager Tutorial

    This tutorial shows how you can manage application contexts.

  • Contact Record Tutorial

    This tutorial shows how you can create, list, update, and delete phone contacts.

  • Contact Setting and SIM Tutorial

    This tutorial shows how you can manage the contact display and the contacts stored on the SIM card.

  • Content Group Tutorial

    This tutorial shows how you can find media items grouped by a given property.

  • Content MIME Type Tutorial

    This tutorial shows how you can map MIME types to file extensions.

  • Display Parameter Tutorial

    This tutorial shows how you can set different display parameters.

  • Location Tracking Tutorial

    This tutorial shows how you can track position, location, and velocity.

  • Metadata Extractor Tutorial

    This tutorial shows how you can extract metadata from a given media file.

  • Notification Tutorial

    This tutorial shows how you can manage notification lists and status.

  • Shortcut Tutorial

    This tutorial shows how you can handle shortcuts and dynamicboxes on the Home screen.

  • Sound Manager Session Tutorial

    This tutorial shows how you can obtain and set basic sound session parameters.

  • Stream Information Tutorial

    This tutorial shows how you can retrieve information from video streams.

  • Subtitle Tutorial

    This tutorial shows how you can insert subtitles to video clips.

  • vCard Tutorial

    This tutorial shows how you can import and export contacts from vCard files.

  • Video Util Tutorial

    This tutorial shows how you can set basic video util parameters.
