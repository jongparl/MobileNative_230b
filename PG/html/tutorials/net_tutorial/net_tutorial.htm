Network Tutorials

These tutorials demonstrate how you can establish, maintain, and monitor the connections required to send and receive data over a network, and manage Wi-Fi and Bluetooth connections.

To learn about the Network API features, see the Network Programming Guide.

Warm-up

Become familiar with the Network API basics by learning about:

  • Bluetooth
    • Bluetooth Tutorial

      Enable Bluetooth, connect to other devices, and exchange data.

  • Connection
    • Connection Tutorial

      Get network connection information, such as default gateway and proxy configuration, IPv4/IPv6 address, cellular connection state, and data transfer statistics.

  • NFC
    • NFC Tutorial

      Activate NFC and get cached NFC messages.

    • NFC P2P Bump Tutorial

      Send and receive NDEF messages using the NFC P2P mode.

  • Wi-Fi
    • Wi-Fi Tutorial

      Manage Wi-Fi connections and monitor the Wi-Fi state.
