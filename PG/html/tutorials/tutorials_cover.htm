Native Tutorials

Tizen native tutorials teach you how to incorporate Tizen native features into your application. The API tutorials demonstrate how you can implement the features of specific APIs within your application.

You can learn much more about Tizen features and functionalities by studying the following tutorials:

  • HelloWorld Tutorial

    Demonstrates how you can create a simple HelloWorld application - thus helping you to get more familiar with the Tizen native application development process and using the Tizen SDK.

  • Application Framework Tutorials

    Demonstrates how you can create the backbone for a Tizen native application.

  • Base Tutorials

    Demonstrates how you can handle the basic functions needed by all native applications.

  • Content Tutorials

    Demonstrates how you can manage common media data types, such as image, audio, and video, search content and content information stored on the device, and download content from servers through HTTP access in a Tizen native application.

  • Context Tutorials

    Demonstrates how you can manage contextual information in a Tizen native application.

  • Location Tutorials

    Demonstrates how you can implement location-related information and services.

  • Messaging Tutorials

    Demonstrates how you can manage message capabilities, such as email, MMS, SMS, and push messages.

  • Multimedia Tutorials

    Demonstrates how you can capture and play or display still images, video, and sounds.

  • Network Tutorials

    Demonstrates how you can establish, maintain, and monitor the connections required to send and receive data over a network, and manage Wi-Fi and Bluetooth connections.

  • Security Tutorials

    Demonstrates how you can create a secure repository for keys, certificates, and sensitive data.

  • Social Tutorials

    Demonstrates how you can manage personal data, such as contacts, schedules, and accounts on the device.

  • System Tutorials

    Demonstrates how you can manage and monitor sensors, system information, runtime information, system settings, and system logs.

  • Telephony Tutorials

    Demonstrates how you can how you can use the Telephony Service to obtain call, SIM, network, and modem information.

  • UI Tutorials

    Demonstrates how you can manage user interface-related features.

  • Web Tutorials

    Demonstrates how you can manage Web browser, pages, and content.

  • Open Source Tutorial

    Demonstrates how you can add functionality to your application using open source code.

Once you have learned the Tizen native basics, you can move on to the advanced tutorials.
