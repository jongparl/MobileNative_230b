Application Framework Tutorials

These tutorials demonstrate how you can create the backbone for a Tizen native application.

To learn about the Application Framework API features, see the Application Framework Programming Guide.

Warm-up

Become familiar with the Application Framework API basics by learning about:

  • Application fundamentals
    • Alarm Tutorial

      Launch an application at a specific time.

    • App Control Tutorial

      Launch other applications through your application.

    • Preference Tutorial

      Manage and save application settings.

  • Badges
    • Badge Tutorial

      Create and modify badges.

  • Bundles
    • Bundle Tutorial

      Create and manage a key-value dictionary.

  • Data control
    • Data Control Tutorial

      Manage map data and SQL data.

  • Message port communication
    • Message Port Tutorial

      Register, unregister, and check the message port and send messages through the it.

  • Notifications
    • Notification Tutorial

      Display notifications on the application screen.

  • Package management
    • Package Manager Tutorial

      Manage and retrieve package information.

  • Service applications
    • Service Application Tutorial

      Access and implement service applications.

  • Shortcuts
    • Shortcut Tutorial

      Create and modify shortcuts and dynamicboxes on the Home screen.
