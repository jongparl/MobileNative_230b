UI Tutorials

These tutorials demonstrate how you can manage user interface-related features.

To learn about the UI API features, see the UI Programming Guide.

Warm-up

Become familiar with the UI API basics by learning about:

  • EFL
    • Application Framework Tutorial

      Implement applications based on the Application Framework.

    • EFL Hello World Tutorial

      Create a simple Hello World application UI using the EFL.

    • Form Tutorial

      Create a basic application that provides a form.

    • Panes Widget Tutorial

      Use a Panes widget in your application.

  • EFL UTIL
    • EFL Optimization Tutorial

      Optimize your Tizen native application.

    • EFL UTIL Tutorial

      Create a notification window and adjust its position.

  • TBM surface
    • TBM Surface Tutorial

      Create and manage the Tizen Buffer Manager Surface.

  • UI widgets
    • Gesture Detector Tutorial

      Implement gestures provided by the EFL library.

    • Creating Graphics Tutorial

      Draw a rectangle or a line image on screen using evas objects available on the EFL widget library.

    • Multipoint Touch Tutorial

      Paint on a canvas using touch gestures, and track touch events.

    • Transit Effect Tutorial

      Implement a variety of EFL animation effects.

    • UI Layout Tutorial

      Compose a UI screen using the layout classes available in the EFL widget library.
