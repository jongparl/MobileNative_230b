Content Tutorials

These tutorials demonstrate how you can manage common media data types, such as image, audio, and video, search content and content information stored on the device, and download content from servers through HTTP access in a Tizen native application.

To learn about the Content API features, see the Content Programming Guide.

Warm-up

Become familiar with the Content API basics by learning about:

  • Download
    • Download Tutorial

      Download content asynchronously from a given URL into the device storage.

  • Media Content
    • Media Content Tutorial

      Save the metadata of the media files on the device.

    • Album Tutorial

      Browse a media album and provide additional information about the album.

    • Bookmark Tutorial

      Bookmark specific parts of a video or audio file.

    • Filter Tutorial

      Filter the results provided by search functions.

    • Folder Tutorial

      Obtain information about media folders and their contents.

    • Media Information Tutorial

      Retrieve information about media content and add it in the media database.

    • Playlist Tutorial

      Access and manage the database containing stored playlists.

    • Tag Tutorial

      Get basic information about existing tags.
