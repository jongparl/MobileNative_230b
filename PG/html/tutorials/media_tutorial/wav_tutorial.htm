WAV Player Tutorial

This tutorial demonstrates how you can play the Waveform Audio File Format (WAV).

API functions and data types are defined in the <wav_player.h> and <sound_manager.h> headers files, which have to be included in all applications that use the WAV Player API.

To learn about the WAV Player API features, see the WAV Player Programming Guide.

This tutorial consists of the following part:

  • Starting and Stopping the WAV Player

    Start and stop playing a WAV file.

Starting and Stopping the WAV Player

To start and stop the WAV player:

  1. Start the WAV player.

    Implement a callback function that is called when the WAV file is no longer being played.

    Declare a variable which keeps the return value of the function. Then invoke the wav_player_start() function to play the WAV file:

    #include <stdio.h>
#include <wav_player.h>
#include <sound_manager.h>

static void _playback_completed_cb(int id, void *user_data) 
{
   const char* path = (const char*)user_data;
   printf("Completed! [id:%d, path:%s]", id, path);
}

void main() 
{
   int wav_player_id;
   wav_player_error_e ret;
   const char* wav_path = "/tmp/test.wav";

   ret = wav_player_start(wav_path, SOUND_TYPE_MEDIA, _playback_completed_cb, (void*)wav_path, &wav_player_id);
}
  2. Stop the WAV player.

    To stop the WAV player, use the wav_player_stop() function with the ID of a given WAV player as an argument:

    void my_stop()
{
   wav_player_error_e ret;
   ret = wav_player_stop(wav_player_id);
}
