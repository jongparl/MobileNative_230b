Multimedia Tutorials

These tutorials demonstrate how you can capture and play or display still images, video, and sounds.

To learn about the Multimedia API features, see the Multimedia Programming Guide.

    • Image Util Tutorial

  • Player

      Record audio.

    • Video Recorder Tutorial

      Record video.

  • Sound Manager
    • Sound Manager Tutorial

  • Tone Player
    • Tone Player Tutorial

  • Video Util
    • Video Util Tutorial

  • WAV Player
    • WAV Player Tutorial

      Play WAV files.
