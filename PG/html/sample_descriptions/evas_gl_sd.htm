Evas_GL Sample Overview

The Evas_GL sample application demonstrates how to implement a rotatable cube on the screen using Evas_GL with the Tizen SDK. The sample shows how to handle polygon geometry, and how to write a simple vertex and fragment shader for the cube.

Evas_GL is used to implement OpenGL rendering on Evas. Evas allows you to specially set up image objects, which act as render target surfaces.

The following figure illustrates the main screen of the Evas_GL.

Figure: Evas_GL screen

The application has only one screen, where you can rotate the cube using your finger. To rotate the cube, drag your finger to the desirable direction.
