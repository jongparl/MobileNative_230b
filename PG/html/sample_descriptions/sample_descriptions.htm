Native Sample Applications

The Tizen platform provides native sample applications, which are included in the Tizen SDK. The sample applications demonstrate the powerful features of Tizen and speed up the learning curve for a new developer. They can be executed directly in the Tizen SDK. The sample applications can be used as a ready library of code for your applications.

To access general information about the sample functionality and the content of the sample source files, click the sample name in the following tables.
