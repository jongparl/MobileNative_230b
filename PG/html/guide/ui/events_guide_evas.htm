Evas Events

Evas events happen on a canvas as a whole. While perfectly usable, this kind of event is too low-level for writing applications and is most often used when writing the graphical toolkit itself.

To register a callback to a canvas, use the evas_event_callback_add() function:

void
evas_event_callback_add(Evas *e, Evas_Callback_Type type, Evas_Event_Cb func, const void *data)

The second parameter defines the Evas event type. The third parameter defines a callback function, such as:

void some_evas_object_event_cb(void *data, Evas *e, void *event_info);

Evas Event Types

The following values are defined in the Evas_Callback_Type enumerator and most of them can be used with the evas_event_callback_add() function:

  • EVAS_CALLBACK_RENDER_FLUSH_PRE

    Rendering is about to be updated on the canvas.

  • EVAS_CALLBACK_RENDER_FLUSH_POST

    Rendering has been updated on the canvas.

  • EVAS_CALLBACK_CANVAS_FOCUS_IN

    Canvas has received the focus.

  • EVAS_CALLBACK_CANVAS_FOCUS_OUT

    Canvas has lost the focus.

  • EVAS_CALLBACK_CANVAS_OBJECT_FOCUS_IN

    Object on the canvas has received the focus.

    Consider using the evas_object_event_callback_add() function and the EVAS_CALLBACK_FOCUS_IN type instead.

  • EVAS_CALLBACK_CANVAS_OBJECT_FOCUS_OUT

    Object on the canvas has lost the focus.

    Consider using the evas_object_event_callback_add() function and the EVAS_CALLBACK_FOCUS_OUT type instead.

  • EVAS_CALLBACK_RENDER_PRE

    Rendering on the canvas is about to start.

  • EVAS_CALLBACK_RENDER_POST

    Rendering on the canvas has finished.
