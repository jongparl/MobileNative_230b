Textblock Objects

Unlike basic text objects, a textblock handles complex text, multiple styles, and multiline text based on tags resembling HTML. These extra features are heavier on memory and processing.

A textblock object can display big chunks of text. The textblock supports many features, such as text formatting, automatic and manual text alignment, and embedding items, such as icons.

A textblock includes the following parts:

  • Text paragraphs
  • Format nodes
  • Cursors

You can use the following markup to format text:

<font_size=50>Big!</font_size>

You can also put more than 1 style directive in a single tag:

<font_size=50 color=#F00>Big and Red!</font_size>
Note
You can use the </font_size> closing tag even though the format also includes color. The first format tag determines the matching closing tag.
