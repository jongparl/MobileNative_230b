Clipping Objects

There are 2 ways to conduct clipping on Evas:

  • Limiting visibility

    An Evas object can be clipped, meaning that its visible area is restricted to the clipper object. A clipper is a rectangle that defines what is visible.

    It is often necessary to show only part of an object. While it is possible to create an object that only contains the visible part, it is often easier to use a clipper.

    Create a clipper by drawing a solid white rectangle (which is the default, so there is no need to call the evas_object_color_set() function) with a position and size to the object displayed. The following code is an example of showing the center half of an Evas object (my_evas_object):

    Evas_Object *clipper = evas_object_rectangle_add(evas_canvas);
evas_object_move(clipper, my_evas_object_x / 4, my_evas_object_y / 4);
evas_object_resize(clipper, my_evas_object_width / 2, my_evas_object_height / 2);
evas_object_clip_set(my_evas_object, clipper);
evas_object_show(clipper);
  • Creating a color layer

    In the previous example, the solid white clipper did not change the color of the clipped object. To change the color of an object, use a clipper that uses a color other than white. Clippers with color multiply the colors of the clipped object. The following code shows how to remove all red from an object:

    Evas_Object *clipper = evas_object_rectangle_add(evas);
evas_object_move(clipper, my_evas_object_x, my_evas_object_y);
evas_object_resize(clipper, my_evas_object_width, my_evas_object_height);
evas_object_color_set(clipper, 0, 255, 255, 255);
evas_object_clip_set(obj, clipper);
evas_object_show(clipper);
