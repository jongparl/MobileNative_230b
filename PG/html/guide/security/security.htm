Security: Providing Security Functions and Secure Repository

The Security API provides basic cryptographic functions, various utility functions (OpenSSL open source library), and information on privileges for user notification.

The main features of the Security API include:

  • Key Manager

    Provide a secure repository for keys, certificates, and sensitive data related to users and their password-protected applications.

  • Privilege Info

    The Privilege Info API provides functions to get information about permissions granted by a privilege.
