Network: Managing Connections and Communication

The Network API contains network- and connectivity-related functionalities, such as Bluetooth and Wi-Fi. The Network API provides functions for establishing, maintaining, and monitoring the connections required to send and receive data over a network. You can also manage Wi-Fi and Bluetooth connections.

The main features of the Network API include:

  • Connection
    • Connection manager

      Get information about the connected network.

    • Connection profile

      Set up and configure network connections.

    • Connection statistics

      Track information, such as sent and received size.

  • Bluetooth
    • Bluetooth adapter

      Set up Bluetooth and discover other devices.

    • Bluetooth device

      Manage bonds with other devices and search for supported services.

    • Bluetooth audio

      Connect to Bluetooth audio devices, such as headset, hand-free, and headphone.

    • Bluetooth GATT

      Discover, read, and modify attributes.

    • Bluetooth HDP

      Manage connections to health devices and exchange data.

    • Bluetooth HID

      Connect to a Bluetooth HID, such as a keyboard or mouse.

    • Bluetooth OPP client and server

      Push and accept objects, such as pictures.

    • Bluetooth socket

      Manage connections to other devices and exchange data.

  • Wi-Fi
    • Wi-Fi manager

      Activate and deactivate a Wi-Fi device, and connect to an access point.

    • Access Point

      Manage network and security information.

    • Wi-Fi monitor

      Monitor Wi-Fi changes.

Note
Within the Network Programming Guide, the term "network" refers to IP network connections. For information on telephony connections, see Telephony Programming Guide.
