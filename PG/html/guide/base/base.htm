Base: Providing Fundamentals

The Base API is the basic module in Tizen native. It is designed to be efficient and reliable, since the functionality of all the other module depends on it in some way. It contains classes for handling a variety of data types, and basic methods often needed in any application.

The main features of the Base module include:

  • I18n

    Enables flexible generation of number or date format patterns.
