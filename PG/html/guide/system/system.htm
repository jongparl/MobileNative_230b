System: Getting System Information and Managing Devices

The System API provides various system information and a functionality to manage attached devices. The main features of the system module include:

  • Device

    The Device API provides functions to control attached devices.

  • Dlog

    Send log messages to a circular log device.

  • Runtime information

    Retrieve memory or storage information in your application, such as allocated and available amounts, by querying runtime information keys.

  • Sensors

    Receive data from various device sensor types.

  • Storage

    Access the accessible parts of the device file system.

  • System information

    Enable your application to retrieve information about fixed platform features or device capabilities by querying system information keys.

    Table: System information categories
    Category Description
    Device ID Device unique ID.
    Platform information Platform or API version, screen coordination system, and CPU and FPU architecture.
    Device capabilities Sensors, networks, connections, and additional components.
