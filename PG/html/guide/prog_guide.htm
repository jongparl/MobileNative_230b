Native Programming Guide

Tizen native programming guide introduces the native APIs used in creating Tizen native applications:

  • Application Framework: Controlling Your Application

    Contains the basis of application development.

  • Base: Providing Fundamentals

    Contains a variety of data types and basic methods often needed in any application.

  • Content: Managing Media Content

    Provides functions and interfaces that allow you to handle content management and content search.

  • Context: Managing Contextual Data

    Provides functions and interfaces that allow you to control user and device information.

  • Location: Using Location Information and Services

    Enables you to add location-based features in you applications.

  • Messaging: SMS, MMS, Email, and Push

    Provides you with access to the device messaging capabilities.

  • Multimedia: Handling Audio, Video, Camera, and Images

    Enables you to integrate audio, video, image processing, and streaming media into your applications.

  • Network: Managing Connections and Communication

    Provides the tools needed for managing network connections and for transferring data to Tizen devices.

  • Security: Handling Keys and Cryptographic Operations

    Provides tools to handle keys and cryptography.

  • Social: Managing Personal Data

    Enables you to create applications that maintain the user's personal data.

  • System: Getting System Information and Managing Devices

    Enables your application to access device-related information and certain device features.

  • Telephony: Handling Phone Call, SIM Card, and Network Information

    Provides call, network, and SIM card information.

  • UI: Creating the Application UI

    Enables you to construct graphical user interfaces.

  • Web: Managing Web Pages and Web Content

    Provides tools for integrating Web browsing features into applications.
