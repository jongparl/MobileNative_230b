Application Framework: Controlling Your Application

The Application Framework API contains the basis of application development. The main features of the Application Framework include:

  • Application Fundamentals

    Manage application state change or system events and control mechanisms that launch other applications:

    • Start or exit the main event loop
    • Register callbacks for application state change events
    • Register callbacks for basic system events
    • Get information about the application
  • Badges

    Create and manage badge icons.

  • Bundles

    Manage bundles and their content.

  • Notifications

    Provide notification functionality for an application.

  • Package Management

    The package manager is one of the native modules of the Tizen application framework. It is responsible for installing, uninstalling, and updating packages, and storing their information. With the package manager, you can also retrieve information related to the packages that are installed on the device.

  • Shortcuts

    Add or remove shortcuts for the Home screen.

Service applications are Tizen native applications with no graphical user interface. They can be very useful in performing activities that need to run periodically or continuously, but do not require any user intervention.

Applications can exchange data using data controls and communicate with each other using message port communication.
