Shortcuts

You can add or remove your shortcut for the Home application.

The Shortcut API allows you to keep shortcuts and dynamicboxes on the home screen.

Figure: Shortcut and dynamicbox

The Tizen Shortcut API requires the http://tizen.org/privilege/shortcut privilege.

To use the shortcut feature:

  • To add a shortcut, use the add_to_home_shortcut() function, which requests the Home application to add a new shortcut.
  • To remove a shortcut, use the add_to_home_remove_shortcut() function, which requests the Home application to remove the specified shortcut.
