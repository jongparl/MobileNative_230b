Tizen Mobile
Native Application Programming

Before using the Tizen development tools and documentation, make sure you are familiar with the license details.

You can create interactive native applications using the Tizen SDK. The IDE contains customized application templates, which provide basic functionality and automatically generate Tizen native source code for your application.

The Tizen mobile native application programming documentation provides all the information you need to create applications for Tizen mobile devices:

  • Basics of Native App Programming

    Introduces the native application model you need to take into account when designing Tizen mobile native applications.

  • Native IDE and Tools

    Provides instructions for using the Tizen native IDE and the tools specific to native application development.

  • Native Application Development Process

    Teaches you how to create a native application.

  • Native Programming Guide

    Provides detailed information on the Tizen native API modules and features.

  • Porting Guide

    Provides detailed information on how to port a 2.2.1 native API application to the 2.3 native API.

  • Native Tutorials

    Teaches you how to incorporate Tizen features in your application.

  • Native API Reference

    Provides descriptions for all Tizen native API modules, classes, and methods.

  • Native Sample Applications

    Provides descriptions for the available native sample applications.
