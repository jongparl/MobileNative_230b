Application Features

To enable filtering for your application:

  1. In the Features tab, click Add and set the mandatory features for the application package.

    The manifest file is updated automatically.

  2. Upload and publish the application package on the Tizen Store.
  3. If a Tizen-powered device requests applications, the store displays a list containing only applications compatible with the user's device.

The following table shows the available requirements for an application package. If you want to check which features are necessary for using a specific API, see Required Feature in the native API Reference.
