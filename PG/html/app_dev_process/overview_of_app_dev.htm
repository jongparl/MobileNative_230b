Native Application Development Process

To develop an application with Tizen:

  1. Plan and design your application.

  2. Create the application project.

  3. Design the application UI and implement code.

  4. Build your application.

  5. Run and debug your application.

  6. Package your application.

You can also develop multiple projects as a combined package.
