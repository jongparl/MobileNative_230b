Developing Multiple Projects as a Package

Tizen supports multi-project applications that combine different types of application templates.

Developing Multi-project Native Applications

You can develop a UI project and a service project as a package.

To develop a multi-project native application:

  1. Create a form-based or tab-based UI application.
  2. Create a service application.
  3. To establish a project reference between the UI and service application:
    1. In the Project Explorer view, right-click the UI application.
    2. Select Properties > Project References.
    3. Select the check box for service application, and click OK.

    In the Project Explorer view, a message appears next to the name of the service application project for all the applications you have added to the package.

  4. Build and run the UI application.

    The service application is built and run automatically while the UI application is built and run.
