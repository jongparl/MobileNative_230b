Installer and SMACK

The following figure describes how the application installer works.

Figure: Installer

The installer applies smack according to the privileges that are specified in the manifest file.

An application can only access the resources that are allowed by the privileges. Privileges are used in the App Store to show the permissions and receive user consent.
