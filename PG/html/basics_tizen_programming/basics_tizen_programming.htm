Basics of Native App Programming

This document describes how to get started with Tizen native application programming. It explains the basics or Tizen native application architecture and life-cycle to help you create Tizen native applications using the Tizen SDK. For more information about developing, building, running, and packaging Tizen native applications, see Native application development process.

Before creating native applications, familiarize yourself with the following:

  • Native application architecture
  • Native application API
  • Non-API Bound Privileges
  • Native application model
  • Developing UI applications with EFL
  • Creating multilingual applications
  • Creating a Hello World application
