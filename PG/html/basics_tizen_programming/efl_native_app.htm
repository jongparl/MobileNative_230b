Using EFL in Tizen Native Applications

To start the Ecore main loop and move your application into the running state, call the app_efl_main() function. The Ecore main loop handles all general events, such as touch, mouse, key, and network events.

int
main(int argc, char *argv[])
{
   ..

   ret = app_efl_main(&argc, &argv, &event_callback, &ad);

   return ret;
}
After your application is running:
  1. Create a window, and organize your widgets inside the window.
  2. Create the application logic code to be called when callbacks or timed events occurs (for example, animators for animations, timers for timeouts).
